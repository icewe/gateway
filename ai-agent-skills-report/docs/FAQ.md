# AI Agent 常见问题FAQ

## Q1: 如何选择合适的模型?

A: 根据任务复杂度
- 简单任务 → GPT-4o-mini / Claude-3-Haiku
- 中等任务 → GPT-4o / Claude-3.5-Sonnet
- 复杂推理 → o1 / Claude-3-Opus

## Q2: 如何降低API成本?

A: 
- 使用缓存
- 压缩prompt
- 选择小模型
- 批量处理

## Q3: 如何提高Agent可靠性?

A:
- 添加重试机制
- 实现错误处理
- 使用验证步骤
- 监控关键指标

## Q4: 如何处理上下文长度限制?

A:
- 摘要旧内容
- 滑动窗口
- 分段处理
- 外部存储
