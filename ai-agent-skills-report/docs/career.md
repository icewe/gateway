## 学习路径

### 阶段 1: 入门 (1-2周)

| 技能 | 资源 | 时间 |
|------|------|------|
| Prompt 基础 | OpenAI docs | 2天 |
| API 调用 | 各平台文档 | 3天 |
| 基本概念 | LangChain 教程 | 3天 |
| 实战项目 | 小工具 | 5天 |

### 阶段 2: 进阶 (1个月)

| 技能 | 资源 | 时间 |
|------|------|------|
| Skills 开发 | Skills.sh | 1周 |
| MCP 集成 | MCP 文档 | 1周 |
| Agent 框架 | LangGraph/Coze | 2周 |

### 阶段 3: 专家 (持续)

- 参与开源项目
- 关注前沿论文
- 构建生产级系统
