# 定时任务

## Cron 表达式

```
┌───────────── 分钟 (0-59)
│ ┌─────────── 小时 (0-23)
│ │ ┌───────── 日 (1-31)
│ │ │ ┌─────── 月 (1-12)
│ │ │ │ ┌───── 星期 (0-6)
│ │ │ │ │
* * * * *
```

## 示例

| 表达式 | 含义 |
|--------|------|
| `0 * * * *` | 每小时 |
| `0 8 * * *` | 每天8点 |
| `0 8 * * 1-5` | 工作日8点 |
| `0 */2 * * *` | 每2小时 |
