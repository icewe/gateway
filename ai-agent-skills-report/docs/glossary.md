# AI Agent 术语表

## A

- **Agent**: 能自主执行任务的 AI 系统
- **API (Application Programming Interface)**: 应用程序接口
- **ART (Agentic Reasoning Tool)**: Agent 推理工具

## B

- **Backoff**: 退避策略（重试间隔递增）
- **Benchmark**: 基准测试

## C

- **Claude Code**: Anthropic 的 CLI AI 工具
- **Coze**: 字节跳动的 Agent 平台
- **Context Window**: 上下文窗口大小
- **Copilot**: GitHub 的 AI 编程助手

## D

- **Dify**: 开源 LLM 应用开发平台
- **Domain Adaptation**: 领域适配

## E

- **Embeddings**: 向量嵌入
- **Evals**: 评估测试

## F

- **Fine-tuning**: 微调
- **Few-shot Learning**: 少样本学习

## G

- **Guardrails**: 安全护栏
- **GPT**: Generative Pre-trained Transformer

## H

- **Hallucination**: 幻觉（AI 生成错误信息）

## I

- **Inference**: 推理
- **In-context Learning**: 上下文学习

## L

- **LangChain**: LLM 应用开发框架
- **LLM (Large Language Model)**: 大语言模型
- **LoRA**: 低秩适配（微调技术）

## M

- **MCP (Model Context Protocol)**: 模型上下文协议
- **Multi-modal**: 多模态

## P

- **Prompt**: 提示词
- **Prompt Engineering**: 提示词工程
- **PR (Pull Request)**: 拉取请求

## R

- **RAG (Retrieval-Augmented Generation)**: 检索增强生成
- **RLHF (Reinforcement Learning from Human Feedback)**: 人类反馈强化学习

## S

- **Skill**: AI Agent 的可复用能力包
- **System Prompt**: 系统提示词
- **Streaming**: 流式输出

## T

- **Temperature**: 随机性参数
- **Token**: 词元（LLM 计算单位）
- **Tool**: 工具（Agent 可调用的函数）

## Z

- **Zero-shot Learning**: 零样本学习
