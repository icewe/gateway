# 国际化 i18n

## 前端 i18n

```javascript
// react-intl 示例
import { FormattedMessage } from 'react-intl';

<FormattedMessage 
  id="greeting" 
  defaultMessage="Hello {name}" 
  values={{ name: 'World' }} 
/>
```

## 后端翻译

```javascript
const i18n = {
  messages: {
    en: { hello: 'Hello', bye: 'Bye' },
    zh: { hello: '你好', bye: '再见' }
  },
  
  t(key, locale = 'en') {
    return this.messages[locale]?.[key] || key;
  }
};

i18n.t('hello', 'zh'); // 你好
```
