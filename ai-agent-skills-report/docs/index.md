# AI Agent Skills & MCP 研究报告

## 简介

本报告调研 AI Agent 能力扩展平台，涵盖 Skills、MCP、开放能力等生态。

## 主要发现

### Skills 生态
- **Skills.sh**: 最大开放市场，TOP Skill 安装量超 33 万
- **Anthropic/Vercel**: 官方 Skills 覆盖前端开发、测试等

### MCP 生态  
- **官方服务器**: 20+ 参考实现
- **MCP Registry**: 社区服务器集中地
- **SDK**: 10+ 语言支持

### 平台
- Coze (字节跳动)、Dify、LangChain 等
