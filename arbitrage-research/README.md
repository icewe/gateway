## 概述

跨交易所套利是指利用不同交易所之间同一资产的价格差异，通过同时买卖来获取无风险收益的交易策略。本文档主要介绍期权（Options）和期货（Futures）市场中的套利方法。

**重要提示**：套利并非无风险，需要考虑交易成本、流动性、滑点等因素。

---

## 基本概念

### 期货套利

- **期货**：约定在未来某一特定时间以特定价格交割的标准化合约
- **基差**：现货价格与期货价格之间的差值
- **价差**：不同月份/交易所期货合约之间的价格差

### 期权套利

- **期权**：赋予持有者在未来以特定价格买入或卖出资产的权利
- **Call（看涨期权）**：买入资产的权利
- **Put（看跌期权）**：卖出资产的权利
- **价差**：同一标的物不同行权价/到期日期权之间的价差

---

## 期现套利

### 原理

当期货价格与现货价格出现较大偏差时，可以通过以下方式套利：

1. **正向套利**（期货溢价）：
   - 买入现货
   - 卖出期货合约
   - 持有至交割获利

2. **反向套利**（反向市场）：
   - 卖空现货
   - 买入期货合约

### 计算示例

```
假设：
- 现货价格：100 USDT
- 3个月后期货价格：105 USDT
- 年化无风险利率：5%

理论期货价格 = 100 × (1 + 5% × 0.25) = 101.25 USDT

实际期货价格 105 > 理论价格 101.25 → 正向套利机会
```

## 跨期套利

### 原理

利用同一交易所不同到期日合约之间的价差波动：

- **牛市价差**：买入近月合约，卖出远月合约
- **熊市价差**：卖出近月合约，买入远月合约
- **蝶式价差**：组合多个到期日的合约

### 适用场景

- 期限结构陡峭或平坦时
- 预期波动率变化时
- 库存周期变化时

## 跨交易所套利

### 跨交易所期货套利

在不同交易所之间寻找同一标的物的价差：

#### 步骤1：识别套利机会

```
交易所A BTC期货（3个月）：45,000 USDT
交易所B BTC期货（3个月）：45,200 USDT
价差：200 USDT

扣除交易手续费+资金成本后仍有收益 → 套利机会
```

#### 步骤2：执行策略

1. 在低价交易所买入
2. 在高价交易所卖出
3. 持有至价差收敛或交割

### 跨交易所期权套利

#### 波动率套利

- 买入低波动率期权
- 卖出高波动率期权
- 对冲Delta风险

#### 套利定价模型

| 因素 | 说明 |
|------|------|
| Black-Scholes | 理论定价 |
| 隐含波动率 | 市场定价 |
| 实际波动率 | 历史数据 |

## 风险因素

### 市场风险

- 价格不利变动
- 流动性枯竭
- 极端事件（黑天鹅）

### 操作风险

- 滑点
- 执行延迟
- 系统故障

### 成本风险

- 手续费
- 资金成本
- 仓储成本（实物交割）

### 监管风险

- 交易所规则变化
- 保证金要求
- 仓位限制
