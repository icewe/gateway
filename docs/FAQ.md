# x402 支付协议 - 常见问题 FAQ

## 基础问题

### Q: 什么是 x402?
x402 是一个基于 HTTP 402 状态码的支付协议，让 AI Agent 可以自动为 API 请求付费。

### Q: 为什么选择 x402?
- 零手续费（只需网络费）
- 即时结算
- 无需账户
- AI 原生设计

### Q: 支持哪些区块链?
- Base (Ethereum L2)
- Ethereum
- Solana
- Aptos (开发中)

## 技术问题

### Q: 如何获取测试网 USDC?
1. 安装 MetaMask
2. 切换到 Base Sepolia
3. 访问 https://coinbase.com/faucet 获取测试 ETH
4. 在测试网 DEX 兑换 USDC

### Q: 支付失败怎么办?
检查清单：
- [ ] 私钥正确
- [ ] 测试网余额充足
- [ ] 网络配置正确
- [ ] Facilitator URL 正确

### Q: 可以用主网吗?
可以。修改 network 参数：
```javascript
// 测试网
network: "eip155:84532"  // Base Sepolia

// 主网  
network: "eip155:8453"  // Base Mainnet
```

### Q: 如何提现收益?
1. 将测试网 USDC 兑换为主网
2. 转到交易所出售
3. 提现到银行

## 安全问题

### Q: 私钥安全吗?
建议：
- 使用环境变量
- 不提交到 Git
- 使用专用测试账户
- 主网用硬件钱包

### Q: 退款机制?
目前 x402 是单向支付。如需退款，需要在应用层实现。

### Q: 欺诈风险?
防护措施：
- 金额验证
- 支付确认
- 审计日志

## 商业问题

### Q: 如何定价?
建议：
- $0.001-$0.01: API 调用
- $0.01-$0.1: 数据查询
- $0.1-$1.0: 复杂计算

### Q: 需要缴税?
请咨询税务顾问。加密货币收入可能需要申报。

### Q: 商业支持?
Coinbase 提供企业级支持服务。

## 对比问题

### Q: vs 传统支付?
| | 传统 | x402 |
|---|---|---|
| 手续费 | 2.9% | <0.1% |
| 结算 | T+7 | 即时 |
| 最小额 | $1 | $0.0001 |

### Q: vs Stripe?
- Stripe: 信用卡支付
- x402: 加密原生支付

### Q: vs micropayment?
传统微支付无法实现的小额支付，x402 可以。
